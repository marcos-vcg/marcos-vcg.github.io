MarkDown

MarkDown Syntax é uma sintaxe usada para padronizar e facilitar formatação de texto na web, utilizada em aplicativos como Slack e GitHub. É uma linguagem de marcação de texto assim comoHTMLmas que utiliza os símbololos para definir a formatação.

JavaScript

JavaScript foi criada inicialmente para dar vida ás páginas WEB. É uma linguagem de script criada para aumentar a interatividade entre um site e seus visitantes. Os códigos emJSsão executados dentro do navegador e dão acesso aos elementosHTMLpor meio doDOM.

HTML

A HTML é uma linguagem de marcação para a criação de sites. Atualmente na versão 5, ela usa a semântica dos elementos para criar o conteúdo de uma página Web. É através dela que criamos a estrutura donde encaixar todos os elementos das páginas WEB. É onde colocamos os textos, imagens, links de redirecionamento, arquivos a serem chamados como por exemplo o CSS, entre outros.

Acesse o Guia de HTML

CSS

Com o CSS, criamos as configurações visuais que um site vai ter, ou seja damos um estilo próprio a ele, a sua própria identidade exclusiva. É através dele que definimos e editamos todos os estilos e personalizamos a cara que o site terá, escolhendo fontes, tamanhos, cores, centralização, espaçamento e qualquer outra configuração de desig. Dessa forma cada projeto se torna únicamente exclusivo de acordo com as preferências visuais de cada projeto.
